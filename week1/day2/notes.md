# Day 2 - Python Basics & Collections

## 1. Printing in Python:

    - `print()` is used to display output
    #Example:
      - print("Hello")
      - print("I don't like 'Apple'")
    
      Use ' ' or " " correctly
    
    => You can use single or double quotes  '' or "" for a string


## 2. Variables:

        - Variables store values
        
        Example:
        a = 10  
        b = 20  
        print(a + b)  → Output: 30
        
        => No need to define type (Python does it automatically)


## 3. What is a Collection?

      -A collection is used to store multiple values in one variable(different datatypes)
      
      Without collection:
      a = 10  
      b = 20  
      c = 30  
      
      With collection:
      numbers = [10, 20, 30]

## 4. List (Most Common Collection)

### What is a List?

      - Stores multiple values  
      - Ordered  
      - Changeable (mutable)  
      - Allows duplicates  
      
      Example:
      numbers = [10, 20, 30]


### Key Points:
      - You can add, remove, or change values  
      - Order matters  
      - Same value can appear multiple times  
      

## 5. Why Use Lists?

        Instead of creating many variables:
        a = 10  
        b = 20  
        c = 30  

## Use:
numbers = [10, 20, 30]
    - Cleaner  
    - Easier to manage
